Pokedex

HTML, CSS, JavaScript

Hosted App

GitHub Repo

This is my first JavaScript application, a Pokedex that displays information about the first 150 Pokemon. It utilizes the PokeAPI to retrieve data and presents it in a card format. Users can click on a card to access more details about a Pokemon and can also search for a specific Pokemon by name or number.
